段落换行：使用两个以上的空格加上回车或者用空行代替
粗体文字 ：文字内容前后加上两个星号或者两个下划线

单行代码：代码前后添加反单引号
System.out.println("helloworld");

代码块：代码前后添加三个反单引号或者4个空格或者1个制表符(Tab键)，可以添加代码语言（添加之后编辑的时候字体会自动变色）

public static void main(String args[]){
    System.out.println("HelloWord");
}
